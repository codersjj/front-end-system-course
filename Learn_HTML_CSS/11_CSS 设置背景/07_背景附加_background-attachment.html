近日，经国务院同意，国家发展改革委、文化和旅游部联合印发《国民旅游休闲发展纲要（2022—2030年）》，旨在进一步优化我国旅游休闲环境，完善相关公共服务体系，提升产品和服务质量，丰富旅游休闲内涵，促进相关业态融合。 《纲要》明确提出要优化全国年节和法定节假日时间分布格局。 第一点现有的假期方案是什么 我们来看看现有的国家公共假期都有什么，不算周六日，仅计算假期一共11天。 (一)新年，放假1天(1月1日)；(二)春节，放假3天(农历正月初一、初二、初三，一般在公历1月21日到2月21日之间)；(三)清明节，放假1天(农历清明当日，一般为公历4月4日到6日之间)；(四)劳动节，放假1天(5月1日)；(五)端午节，放假1天(农历端午当日，一般在5月27日到6月26日之间)；(六)中秋节，放假1天(农历中秋当日，一般在9月8日到10月7日之间)；(七)国庆节，放假3天(10月1日、2日、3日)。 有这样几个特点，A公历假期和农历假期互相掺和，B上半年五个下半年两个苦乐不均，C除了国庆和春节三天以外其他都是一天，D周六日结合起来国庆春节七天其他基本就是三天。 第二点假期改革调整的方案 原则上就是11天公众假期不改变，照顾到平时出行和假期出行的错峰，尊重实际情况，实事求是。建议改革假期方案如下： 1取消元旦，五一，中秋节三天假期； 2增加到春节，端午节里面增加一天； 3增加一个新的节日——冬至节。
