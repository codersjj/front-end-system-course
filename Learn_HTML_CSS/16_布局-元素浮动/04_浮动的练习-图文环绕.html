20年前的五一假期， 我只身出发去爬华山。到山脚下已是黄昏，跟着众人一起登山，很快天就黑了。脚下有亮光，四周却黑幽幽的，山路上的灯，璀璨晶莹，明明灭灭，使蜿蜒的山路如忽隐忽现的巨大游龙。黑夜里爬山，心无旁骛，专心攀爬，不知爬了多久，到达东峰顶。睡了几个小时，起来观日出。看着那金色火球从绚烂朝霞中喷薄而出，冉冉升起，万丈金光辉映天空，游客们大声欢呼，满身疲惫霎时一扫而光。欣赏过日出后，又游玩了长空栈道，鹞子翻身等著名景点。在又一个黄昏到来时步行下山，但已记不清是哪条路了。二十年后的八月，我和儿子一起爬华山。清晨6:50登上玉泉宫，开启征程。起初山路宽阔平缓，绿树夹道，清凉幽静。旁边山涧溪水淙淙，山林之中鸟鸣蝉噪，切身体会到“蝉噪林逾静，鸟鸣山更幽”的佳景。到五里关后，山路变陡变窄，我们小心谨慎，缓步前行。走了许久，一段近乎垂直的台阶赫然悬在眼前，恰如一架自高空垂下的天梯，召唤人们由此进入仙境。原来这就是华山著名的险境之一：千尺㠉。欲攀爬，须手脚并用。我们全神贯注，小心翼翼，轻手轻脚，努力爬了上去。过了千尺㠉之后，又跨越了另一极为险峻之处——老君犁沟。征服了这两处天险之后，我们勇气倍增，乘兴前行，于11点到达北峰顶。 站在北峰之巅，极目四望，近处危峰耸立，林木葱茏；远处青山隐隐，云雾弥漫。不少游客在此拍照留念。我们找个阴凉处吃喝休息。约半小时后，继续向西峰进发。西峰之路虽也崎岖坎坷，但总体还算易行。有段平缓之处，秀木夹道，浓荫蔽日，行走其间，轻快欢畅。旁边有数棵百年以上的参天巨树，恰如神威将军，率领着一众绿意葱葱的兵士们，守卫着秀美华山，欢迎着热爱河山的游客。该段路的最险峻之处，莫过于金锁关前的苍龙岭。其势如一条腾空而起的巨龙，左侧缓如鱼背，右侧绝壁万丈。攀爬其上，如履刀刃，心惊胆寒。据传韩愈当年行至此处，吓得大哭，投书求救。但我们既已翻越了千尺㠉和老君犁沟，就不会被它难倒，沉着冷静，抓牢栏杆，慎步缓行，不久就爬到金锁关。稍事休息后，继续在跌宕起伏曲折蜿蜒的山路上爬高走低挥洒汗水，最终到达西峰顶。西峰顶和北峰顶一样壮阔秀美。游目驰怀，心旷神怡，霎时体会到“会当凌绝顶，一览众山小”的万丈豪情。至此，我们的脚步已踏过了北峰中锋西峰，游历了七个多小时。因第二天还有其他安排，无法再攀登东峰南峰，我们就坐索道下山了。巍巍华山，雄伟奇险。奋力登攀，倾心游览。锻炼体魄，磨练意志。美哉此行，大乐其中！
